> Requirements
- A 64-bit Windows, GTK-based Linux, or macOS operating system
- A Java 8 (or above) Runtime Environmemt
  Java 8.191 is recommended - visit https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

> How to install?
- Simply extract the JEB archive contents to a directory of your choice.

> How to set up and use the floating licensed builds?
Tutorial at https://www.pnfsoftware.com/jeb/manual/floating/

> How to execute?
- On Linux, open a terminal and type: ./jeb_linux.sh
- On MacOS, open a terminal and type: "./jeb_macos.sh
- On Windows, open a terminal and type: jeb_wincon.bat
