# Android

## Jeb

1. 进入`jeb`文件夹
2. 打开`./bin/keygen GUI.jar`
3. 再打开`./jeb_wincon.bat`
4. 点击`Manual Key Generation`
5. 点击GUI的`Generate key`，并且复制
6. 粘贴，确定
7. OK，Continue
